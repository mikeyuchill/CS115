Note to PHP Users

This build cannot load any PHP Apache module (e.g. php7apache2_4.dll). For PHP use mod_fcgid (included). To use mod_fcgid for php edit the file conf/extra/httpd-fcgid.conf and uncomment the Include line for fcgid at the bottom of conf/httpd.conf.
